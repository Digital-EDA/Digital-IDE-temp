## 详细说明：

在vlog中分有两种依赖形式 
1. 分module文件依赖，该依赖取决于解释器的自动检索。
   【注】: 在存在同名module的时候很大程度上会导致依赖出错，因此不建议使用同名module，或者在同名的时 候使用 `include 依赖方式
   
2. 使用 include 引用文件的方式进行模块间的依赖，该方式可以精确定位依赖项。
   【注】: 在使用 `include 依赖方式时该插件会自动去除对于include文件中有同名的其他文件。
